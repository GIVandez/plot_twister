Project API Test

1. Создать проект (POST /api/user/createProject)

2. Получить проекты пользователя (GET /api/users/{user_name}/loadInfo)

3. Обновить название проекта (POST /api/user/updateProjectInfo)

4. Удалить проект (DELETE /api/user/deleteProject)

5. Удалить сценарий проекта (DELETE /api/user/deleteScript)

6. Удалить раскадровку проекта (DELETE /api/user/deleteFrames)

7. Связать кадр и страницу (POST /api/frame/connectFrame)

8. Разорвать связь кадра и страницы (POST /api/frame/disconnectFrame)
